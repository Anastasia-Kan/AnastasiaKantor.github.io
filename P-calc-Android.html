Pressure calculator for Android

by Anastasia Kantor

Pressure calculator for Android

My first app on Google Play: A scientific tool for pressure calculation in a diamond anvil cell using ruby fluorescence and diamond Raman spectroscopy. The app supports different scales and a temperature correction.

See on GitHub Check it out on Google Play

    My first app on Google Play

After such a mobile application became available for Apple devices so was it a right time to develop one for Androiod. Here the same idea and the same app-flow was used and the same features as for Apple devices were integrated.

Two tabs - Ruby and Diamond - give the possibility of calculating pressure inside a diamond anvil cell either from ruby fluorescence or from a Raman spectrum from diamond. For the first case one can choose between three different calibrations: according to Shen et al., 2020 or Mao et al., 1986 - hydrostatic or non-hydrostatic conditions. As for the Raman from diamond, one can use either Raman peak from the center of the diamond culet or from a piece of diamond inside a sample chamber.

The user can either swipe between tabs or just tap on a desired one. In addition there is an information fragment for each tab that can be reached by tapping the Info button and containing the references for every calibration used in the application.

Compared to my Swift project this one has a better structure: view models were used for each fragment as well as helper files that include all the long equations, constants and functions that should not be neither in fragments nor in view models.

As a result I have not only a functional application, but also a clean code, that would be much easier to change in case I would add new features.
